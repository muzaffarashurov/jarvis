# NON_GOALS.md

# What Jarvis Is NOT

Jarvis is NOT:

- another chatbot
- another LLM
- a wrapper around ChatGPT
- tied to one AI provider
- a prompt collection
- provider-centric
- a monolithic application

---

# What Jarvis IS

Jarvis IS:

- an AI Operating System
- a workflow orchestrator
- provider-independent
- capability-driven
- modular
- extensible
- future-proof

---

# Architecture Principles

Always prefer:

- loose coupling
- modularity
- reusable components
- provider independence
- capability routing
- extensibility

Avoid:

- duplicated logic
- provider-specific business logic
- hardcoded providers
- monolithic modules
- AI-specific architecture
